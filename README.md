# Water-Save Parallel Builds

Isolated HTML prototypes, build experiments, and iteration artifacts for the Water-Save project.

Each subfolder or file represents an independent build/prototype that was developed in parallel before being merged into the main app.

## Contents
- `*.html` — standalone HTML prototypes (terrain viewer, flow heatmap, soil cross-section, etc.)
- `*.png` — visual iteration screenshots
- `audit*/` — pipeline audit scripts and results
- `*.py` — standalone build/check scripts

## Workflow
Build experiments here first, screenshot and validate, then port to main app.
